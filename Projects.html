Projects

Project 1: AWS Migration

This project involved migrating a legacy on-premises application to AWS. The migration included setting up EC2 instances, configuring VPCs, and using RDS for the database. The project improved the application's scalability and reduced operational costs.

Project 2: Serverless Application

This project focused on building a serverless application using AWS Lambda, API Gateway, and DynamoDB. The application processes user data and stores it in DynamoDB. This architecture reduces the need for server management and scales automatically with usage.

Project 3: AWS CI/CD Pipeline

This project implemented a continuous integration and continuous deployment (CI/CD) pipeline using AWS CodePipeline, CodeBuild, and CodeDeploy. The pipeline automates the testing and deployment of code changes, ensuring rapid and reliable releases.
